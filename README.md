# Alertes Magento
Module basique d'Alertes pour magento.
Ce module vou permets d’envoyer des alertes via emails aux administrateurs ou aux gestionnaires du site via des requêtes sql.
Quelques exemples :

    - Nombre minimal de commandes passé sur le site à X h
    - Commandes ayant utilisé le coupon X
    - Coupons de réductions arrivant à échéance dans X jours
    - Vérification des produits qui ne sont bientôt plus en stock
    - etc…
